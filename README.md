# Wayland VNC Client

## Introduction
This is a work-in-progress implementation of a Wayland native VNC client.
Expect bugs and missing features.

## Runtime Dependencies
 * aml
 * libvncclient
 * libwayland
 * libxkbcommon

## Build Dependencies
 * GCC/clang
 * meson
 * ninja
 * pkg-config
 * wayland-protocols
